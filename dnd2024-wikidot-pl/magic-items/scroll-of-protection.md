Źródło: Podręcznik Mistrza Gry

_Zwój, rzadki_

Każdy _Zwój ochrony_ działa przeciwko istotom wybranego typu, który wybiera MP albo ustala losowo, rzucaj na poniższej tabeli.

| 1k100 | Typ istoty |
| --- | --- |
| 01–10 | aberracje |
| 11–15 | bestie |
| 16–20 | niebiańczycy |
| 21–25 | konstrukty |
| 26–35 | smoki |
| 36–45 | żywiołaki |
| 46–50 | humanoidy |
| 51–60 | fey |
| 61–70 | czarty |
| 71–75 | olbrzymy |
| 76–80 | monstra |
| 81–85 | szlamy |
| 86–90 | rośliny |
| 91–00 | nieumarli |

Odczytanie zwoju akcją magiczną tworzy emanację o promieniu 1,5 metra wychodzącą od ciebie. Przez 5 minut istoty wybranego typu nie mogą wejść w obszar ani wpływać na nic w jego obrębie. Jeśli jednak poruszysz się tak, że istota wybranego typu znalazłaby się w obszarze, efekt się kończy.

Akcją magiczną istota w promieniu 1,5 metra od emanacji może spróbować ją przełamać, wykonując rzut obronny na Charyzmę o ST 15. Przy udanym rzucie przestaje być pod wpływem emanacji.
