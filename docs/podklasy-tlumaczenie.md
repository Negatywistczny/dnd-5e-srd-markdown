# Słownik nazw podklas (EN → PL)

Oficjalne i rekomendowane polskie nazwy podklas dla `dnd2024-wikidot-pl/subclasses/`.

**Kontekst:** polskiego Podręcznika Gracza 2024 **nie ma** — ten projekt ma go zastąpić. Nazwy podklas z edycji 2024 nie mają oficjalnego wydawcy PL; ustalamy je w ramach kuracji (oznaczenie **proj.**).

## Źródła (w kolejności wiarygodności)

| Skrót | Źródło |
| --- | --- |
| **PG** | [`sources/5e-SRD_v1.0.md`](../sources/5e-SRD_v1.0.md) — oficjalne PL SRD 5.1 |
| **PG14** | [TESGIR — PG 2014](http://tesgir.pl/dnd-5ed-rasy-klasy-i-pochodzenia/) — wyd. Galaktyki |
| **Xan** | [Rebel — Xanathar](https://www.wydawnictworebel.pl/pages/przewodnik-po-wielu-podklasach-1586.html) |
| **Foundry** | `lang-pl-dnd5` — kompendium Foundry VTT |
| **srd** | `srd-5.2.1/pl/classes.md` — materiał wejściowy (korygujemy błędy) |
| **proj.** | Kuracja tego repozytorium — podklasy nowe/zmienione w 2024 |
| **UA** | Materiał UA / dodatkowy — brak oficjalnego PL |

Powiązane: [terminologia mechanik](terminologia-tlumaczenie.md), [czary](czary-tlumaczenie.md).

## Konwencje projektu

- Klasy: Wojownik, Mnich, Paladyn, Łowca, Łotr, **Zaklinacz**, Czarownik, Mag, Wynalazca (UA).
- Sorcerer → **Zaklinacz** (PG). „Czarnoksiężnik” w starym tłumaczeniu SRD to błąd.
- Tytuł pliku (H1) = ustalona nazwa podklasy po polsku.
- Wpisy **proj.** można zmienić po dyskusji — nie czekają na książkę, której nie będzie.
